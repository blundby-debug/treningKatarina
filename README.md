# Katarina — Treningslogg (installasjon)

1. Nytt GitHub-repo (gratis) — f.eks. `katarina-trening`, Public.
2. Add file → Upload files → dra inn alle filene i denne mappen.
3. Settings → Pages → Deploy from branch → main / (root) → Save.
4. Åpne URL-en i Safari på telefon → Del → Legg til på Hjem-skjerm.

Lagring er localStorage, egen prefix (kt:), kolliderer ikke med andre PWA-er på samme enhet.
Eksporter jevnlig som backup.
